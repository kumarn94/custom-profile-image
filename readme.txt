== Description ==

The Custom Profile Image Plugin for WordPress lets users upload their own profile pictures or use their Gravatar. It allows easy image management and seamless switching between custom and Gravatar images. The plugin also integrates with WordPressâ€™s default get_avatar() function for easy access to profile images.

== Installation ==

1. Upload [custom-profile-image](https://github.com/kumarn94/custom-profile-image/archive/refs/heads/main.zip) to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
